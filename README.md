## Installation
To get started with this project, follow these steps:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/nadahossamismail/plantiful.git
    cd plantiful
    ```
    
2. **Install dependencies:**
    ```bash
    flutter pub get
    ```

3. **Run the app:**
    ```bash
    flutter run
    ```
